## 🚀 GitHub Pages Deployment

1. In your repository, go to **Settings** > **Pages**  
2. Under "Source", select the `main` branch and `/root` folder  
3. Click **Save**  
4. Your site will be published at `https://<username>.github.io/<repo-name>/`
